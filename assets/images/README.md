# Payment Gateway Logo Images

This directory should contain logo images for payment gateways:

- `paypal-logo.png` - PayPal logo (recommended size: 200x80px)
- `stripe-logo.png` - Stripe logo (recommended size: 200x80px)

## Note

The payment gateway buttons will show text fallbacks if images are not present. You can add your own logos or download official brand assets from:

- PayPal: https://www.paypal.com/us/webapps/mpp/logos-buttons
- Stripe: https://stripe.com/newsroom/brand-assets

## Image Requirements

- Format: PNG with transparent background
- Max width: 200px
- Max height: 80px
- Optimize for web (compress to reduce file size)
